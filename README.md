# Smart-Plantation-System
An IoT-based automated plant care system using ESP32 and Blynk

## 🌿 About The Project
[cite_start]The Smart Plantation System automates tasks that would otherwise require constant human involvement[cite: 21]. [cite_start]By employing IoT tools and various sensors, it constantly monitors plant health and growing conditions to ensure optimal survival and growth[cite: 22, 23].

**Institution:** Sri Lanka Institute of Information Technology (SLIIT)
**Course:** IT1040 - Fundamentals of Computing
**Team:** Group 05

### 🎯 Key Objectives
* [cite_start]Constantly check temperature, humidity, and soil moisture using specifically placed sensors[cite: 63].
* [cite_start]Automatically control water pumps, fertilizer release, and fans to restore optimal plant conditions[cite: 64].
* [cite_start]Monitor plant height using ultrasonic sensors to determine when the plant is ready to be moved outdoors[cite: 62].
* [cite_start]Send weekly plant health reports as graphs via the Blynk mobile app[cite: 65, 98].

## 🛠️ Hardware Components
* [cite_start]**ESP32:** Acts as the central controller for all connected hardware[cite: 89].
* [cite_start]**Sensors:** Soil moisture sensor, DHT11 sensor (for temperature and humidity), and Ultrasonic sensor (for plant height)[cite: 90].
* [cite_start]**Actuators:** Water pumps and fans[cite: 90].
* [cite_start]**Display:** 16x2 I2C LCD display to show plant height[cite: 80, 97].
* [cite_start]**Storage & Power:** SD Card and Power Supply[cite: 76, 83].

## 💻 Software & Technologies
* [cite_start]**Arduino IDE:** Used to write the C++ code to control the ESP32 board and automate plant maintenance.
* [cite_start]**Blynk App:** Used to design the mobile application, connect IoT devices, and generate weekly data reports[cite: 93, 98].

## 🧪 Evaluation & Testing
To ensure reliability, we employed several evaluation methods:
* [cite_start]**Functional Testing:** Individual sensor testing for calibration, followed by integrated testing to verify system communication[cite: 104, 105, 106].
* [cite_start]**Accuracy Testing:** Comparing sensor readings against manual measurements taken with traditional tools[cite: 109].
* [cite_start]**User Feedback:** Having selected users interact with the system to ensure it is user-friendly and useful[cite: 110].
